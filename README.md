# Statistical and Spectral Analysis of Turbulent Flows Using DNS and PIV Data.

* Analysed isotropic turbulence DNS data on a $1024^3$ grid, computing energy spectra, velocity correlations, and structure functions to verify $k^{-5/3}$ inertial-range scaling and scale-dependent turbulent behaviour.
\item Processed Particle Image Velocimetry (PIV) data for a compressible jet flow to extract mean and fluctuating velocity fields, identifying shear layers, shock locations, and regions of turbulence intensification using RMS velocity analysis.
